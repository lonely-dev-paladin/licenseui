About Authra Panel

A license key management system for software distribution — built with a self-service admin marketplace, multi-device license support, and a full audit trail.

🔑 What Admins Can Do

  • Generate license keys — single-device, or multi-device with up to 1,000 device slots per key
  • Ban, unban, extend, or permanently delete any key they've issued
  • Reset a bound device so a key can be activated on a new machine
  • View all issued keys, organized separately by single-device vs. multi-device
  • View and clear their own audit log of every action they've taken

👑 What Superadmins Can Do

  • Everything an Admin can do, plus:
  • Create new admin accounts directly
  • Review, approve, or reject self-service admin purchase requests
  • View every admin account on the platform, with live key and audit-log counts
  • Permanently terminate an admin or moderator account and all of their data

✨ Notable Features

  • A license key's countdown only starts once the first device actually activates it — not at creation
  • Self-service "Become an Admin" flow — buy a week, month, or lifetime seat via GCash
  • Full audit trail covering every key and admin-management action
  • Companion Android app with offline grace period, English/Tagalog/Cebuano language support, light/dark themes, and mini-games

⚙️ Built With

Python (Flask) PostgreSQL Supabase Render Redis (optional) HTML / CSS / JS AndLua+ (Android)
